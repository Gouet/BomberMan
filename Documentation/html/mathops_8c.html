oc_ilog32 - Integer binary logarithm of a 32-bit value. : A 32-bit value. Returns floor(log2(_v))+1, or 0 if _v==0. This is the number of bits that would be required to represent _v in two's complement notation with all of the leading zeros stripped. The OC_ILOG_32() or OC_ILOGNZ_32() macros may be able to use a builtin function instead, which should be faster.
