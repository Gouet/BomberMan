io::path irr::io::SFileListEntry::FullName

The name of the file including the path.

If this is a file or folder in the virtual filesystem and the archive was created with the ignoreDirs flag then it will be the same as Name.

u32 irr::io::SFileListEntry::ID

The ID of the file in an archive.

This is used to link the FileList entry to extra info held about this file in an archive, which can hold things like data offset and CRC.
