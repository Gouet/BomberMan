Detailed Description

Triangle Selector for the TerrainSceneNode.

The code for the TerrainTriangleSelector is based on the GeoMipMapSelector developed by Spintz. He made it available for Irrlicht and allowed it to be distributed under this licence. I only modified some parts. A lot of thanks go to him.
