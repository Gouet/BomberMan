40 /*Accumulates various weighted sums of the measurements.
41  w -> weight
42  s -> SATD
43  q -> log quantizer
44  r -> rate (in bits)
45  d -> RMSE
46  All of the single letters correspond to direct, weighted sums, e.g.,
47  w=sum(w_i), s=sum(s_i*w_i), etc.
48  The others correspond to central moments (or co-moments) of the given order,
49  e.g., sq=sum((s_i-s/w)*(q_i-q/w)*w_i).
50  Because we need some moments up to fourth order, we use central moments to
51  minimize the dynamic range and prevent rounding error from dominating the
